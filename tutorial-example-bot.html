    Hi, in this tutorial we will be creating a simple bot that will respond to a user's message with a "Hello World!" message.

    Installation

    Use the package manager npm to install Guilded Bot.

    npm install guilded-bot

    Usage/Examples

    const Guilded = require("guilded-bot");
const client = new Guilded.Client("Your-Token-Here");

client.on("ready", () => {
  console.log("I am ready!");
});

client.on("message", (message) => {
  if (message.content === ">ping") {
    message.channel.send("Hello World!");
  }
});

client.login();
